### Simple shortcode
```
[youtube]
```

### Shortcode with attributes
```
[youtube
    id="123" --> Attributes
    width="640" --> Attributes
    height="480" --> Attributes
]
```

### Shortcode with content
```
[youtube]
    My Caption
[/youtube]
```

### Shortcode nested with attributes and content
```
[youtube_container]
    [youtube
        id="123" --> Attributes
        width="640" --> Attributes
        height="480" --> Attributes
    ]
[/youtube_container]
```
